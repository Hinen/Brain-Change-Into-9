게임 명 : 뇌를 9로 바꾸는 변환기
Created By : 박수빈
SourceCode : https://github.com/Hinen/Brain29

1. 본 게임은 Python 3.7.3으로 제작되었습니다. 
	따라서 Python 2.x의 버전으로는 게임이 실행되지 않습니다.

2. 본 게임의 사운드를 듣기 위해선 pip 설치 및 환경 변수 등록이 필요합니다.
	Python 3.x를 설치하면 자동으로 pip가 설치되어 윈도우 환경 변수에 등록되지만 
	만약 pip가 깔려있지 않으시다면 설치하시고, 환경 변수를 등록해야만 사운드가 출력됩니다.

3. 본 게임에서는 "휴먼 엑스포" 폰트를 사용합니다.
	윈도우에 기본적으로 설치되어있는 폰트입니다.
	설치가 안 되어있다면 폴더에 있는 HMFMMUEX_1.ttf 파일로 폰트를 설치하는 것을 권장합니다.
	
4. 본 게임은 숭실대학교 미디어경영학과 19학년도 컴퓨팅적 사고 과목의 과제입니다. 

5. 본 게임을 실행하기 위해서는 Main.py 파일을 실행해주세요.
